一、VS快捷键

Crtl+d 选择多个相同词修改
Shfit+Alt+up 向上复制当前行
Crtl+Alt+up 添加多个光标
Crtl+h 全局替换
Crtl+g 快速跳转到某一行
Shfit+Alt+鼠标 选中某一个区块

Crtl+/ 注释
Shfit+Crtl+/ 块注释(自定义修改过)

二、html常用标签

2.1 标题标签

<h1> <h2> <h3> <h4> <h5> <h6>
6个等级标题，依照重要性递减，文字加粗一行显

2.2 段落标签

1.文本在一个段落中会根据浏览器窗口大小自动换行 2.段落和段落之前有空隙

2.3 换行

<br/>

1.单标签
2.<br /> 或者是 <br>只是简单开始新的一行，还在该段落中，跟段落不同，段落之间会插入一些垂直的间距

2.4 文本格式化：加粗，倾斜

标签语义：突出重要性，比普通文字更重要

2.5 div和span标签

没有语义，用来布局，只是一个盒子，用来装内容
<div></div>

div是division的缩写，表示分割、分区
span意为跨度、跨距

1.一个div独占一行，大盒子
2.一行可以有多个，小盒子

2.6 图像标签和路径

单标签
<img src=””/>
src是标签的必须属性，指定图像路径和文件名

高度和宽度属性一般改一个，另一个会等比例缩放。

图像标签属性注意点：

  • 图像标签可以有多个属性，必须在标签名后面
  • 属性不分顺序，标签名和属性、属性与属性之间以空格分开
  • 属性采取键值对形式，key = “value”

路径


相对路径:引用文件所在位置为参考
绝对路径：+ 从盘符开始，绝对路径用反斜杠\，相对用斜杠/,”D:\web\img\logo.gif”
+ 或用完整的网络地址 http://www.itcast.cn/images/logo.gif“

超链接

<a href = “跳转目标” target = “目标窗口的弹出方式”>文本或图像

链接分类：

1.外部链接
2.内部链接：网站内的页面之间链接，直接写html名即可
3.空链接：href = “#”
4.下载链接：href = “xxx.zip”，直接将exe文件或压缩包名写入即可
5.网络元素链接:图片、音频、视频作为链接入口，将图片放入a标签内即可
6.锚点链接：点击链接，可以快速定位到页面中某个位置

  • 在href属性中，设置属性值为**#名字**的形式，如<a href = “#two”>第二集
  • 找到目标位置标签，添加id属性 = 刚才的名字，如<h3 id = “two”>第二集介绍

2.7 注释

<!– hhhh –>
快捷键：Crtl+/

2.8 特殊字符

重点记住：空格、大于、小于号

三、表格标签

3.1 表格标签的基本使用

表格的作用

表格用于展示数据,可以让数据显示非常规整

表格的基本语法

  • <table>用于定义表格的标签
  • <tr>定义行，嵌套在<table>标签中。
  • <td>定义表格中的单元格，嵌套在<tr>标签中。
  • 字母td指的是表格数据table data,即单元格的内容，注意：单元格内可以放任何元素，文字链接、图片都可以

3.2 表头单元格标签

th是table head缩写
<th>与td的区别：加粗、居中显示

3.3 表格属性

表格标签的属性实际开发中不常用，通常用CSS来设置
目的有两个：

  • 记住相关单词，后面CSS会使用
  • 直观感受表格的外观形态

align是浏览器中对齐方式
cellpadding是单元格边框和单元格内的文字的距离
cellspacing是单元格与单元格之间的距离

3.4 表格结构标签

<thead>标签：表格的头部区域，内部必须有<tr>标签，一般位于第一行
<tbody>标签：表格的主体区域，主要用于放数据本体
两标签都放在<table>标签中

3.5 合并单元格

合并单元格方式

  • 跨行合并：rowspan = “合并单元格的个数”
  • 跨列合并：colspan = “合并单元格的个数”
